Run solr
1. run cmd
2. change directory to bin of solr-8.0.0  (".\IR_Q1-Q3\solr-8.0.0\solr-8.0.0\bin")
3. type "solr start"

To run the website
1. open IR_SearchEngine in Pycharm
2. type "python manage.py runserver" in terminal
3. navigate to "localhost/searchengine"

Crawled data are located in "IR_Q1-Q3\IR_SearchEngine\crawl"
